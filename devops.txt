# DevOps Learning Notes 
 
## What is DevOps? 
DevOps is a set of practices that combines software development (Dev) and IT operations (Ops). It aims to shorten the development lifecycle and provide continuous delivery with high software quality. 
 
## Key DevOps Tools: 
- **Version Control**: Git, GitHub, GitLab, Bitbucket 
- **CI/CD Pipelines**: Jenkins, GitHub Actions, GitLab CI/CD 
- **Configuration Management**: Ansible, Chef, Puppet 
- **Containerization**: Docker, Podman 
- **Orchestration**: Kubernetes, OpenShift 
 
## DevOps Lifecycle: 
1. **Plan** - Define project requirements 
2. **Develop** - Write and manage code 
3. **Build** - Automate build processes 
4. **Test** - Implement automated testing 
5. **Release** - Deploy software efficiently 
6. **Monitor** - Track application performance
